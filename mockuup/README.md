# Game Bling Mockups

Static mobile-first HTML mockups for the World Cup prediction room app.

Open `index.html` in a browser and use the screen buttons on the left, or the phone bottom navigation, to move through:

- Join/create room
- Home urgency feed
- Prediction entry
- Live match detail
- Room and global leaderboard
- Room invite/share
- Profile and avatar selection

These files are design mockups only. They do not connect to Supabase or any live football API.
