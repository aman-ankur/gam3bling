Implementation direction

  • After save, route stays on the match page but starts in a compact saved state: one-line receipt first, friends' predictions immediately below.
  • The detailed prediction form moves into an expandable edit panel, so changing a prediction is still one tap before kickoff.
  • Home should become personalized when a browser session exists: joined rooms, current open predictions, live/scored alerts, then the generic join/create sections.
  • The room page should become a room hub: current fixtures first, then room leaderboard, then recent history. Later it can support streaks, settlement tracking, and recap/commentary.
